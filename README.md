# Task 8 – Model Deployment

## Project Overview

This repository contains the deployment phase of the Customer Churn Prediction project.

## Objective

To deploy the trained Logistic Regression model using Flask and expose it as a REST API for real-time customer churn prediction.

## Features

- Loads the trained machine learning model
- Loads the preprocessing pipeline
- Accepts customer information in JSON format
- Predicts customer churn
- Returns prediction as JSON response
- Tested using a Python client script

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Pickle
- Requests
